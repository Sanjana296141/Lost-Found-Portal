# Lost & Found Portal 🎒🔍

A modern full-stack Lost & Found web application developed using Node.js, Express.js, MySQL, HTML, CSS, JavaScript, and EJS.

This platform helps students/users report lost or found items, search for items, communicate through chat, and manage claims efficiently through a user-friendly dashboard and admin panel.

---

# 🚀 Features

## 👤 User Features
- User Registration & Login Authentication
- User Dashboard
- Upload Lost Items
- Upload Found Items
- Upload Item Images using Multer
- Search & Filter Items
- Item Details Page
- Chat System Between Users
- Notifications System
- Profile System
- Edit/Delete Uploaded Items
- Responsive UI Design
- Dark Mode UI
- Recent Uploads Section

---

## 👑 Admin Features
- Admin Login Panel
- Admin Dashboard
- Total Users & Total Items Analytics
- Manage Users
- Manage Uploaded Items
- Edit/Delete Items
- Search & Filter Tables
- Notification Panel
- Activity Logs
- Animated Dashboard UI
